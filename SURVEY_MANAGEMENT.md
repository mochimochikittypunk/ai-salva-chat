# 12月アンケート機能 管理手順書

このファイルは、2025年12月限定のアンケート機能を管理（無効化・削除）するための手順書です。
以下のテキスト（プロンプト）をコピーしてAIアシスタントに送信するだけで、設定変更や削除が実行できるように記述されています。

## 1. アンケートを無効化する（推奨：1月になったら実行）

アンケート機能を停止しますが、コードは残します。またいつでも再開可能です。

**▼ AIへの指示（以下のテキストをコピーして送信してください）**
```text
src/app/api/chat/route.js の `ENABLE_DECEMBER_SURVEY` フラグを `false` に変更して、アンケート機能を無効化してください。
また、src/app/page.js のデフォルトメッセージに含まれる「（期間限定でアンケートも実施中！...）」という文言も削除してください。
```

## 2. アンケートを有効化する

停止していたアンケート機能を再開します。

**▼ AIへの指示（以下のテキストをコピーして送信してください）**
```text
src/app/api/chat/route.js の `ENABLE_DECEMBER_SURVEY` フラグを `true` に変更して、アンケート機能を有効化してください。
また、src/app/page.js のデフォルトメッセージの末尾に「（期間限定でアンケートも実施中！『アンケート』と送ってね）」という文言を追加してください。
```

## 3. アンケート機能を完全に削除する（コードのお掃除）

アンケート機能が完全に不要になり、コードをきれいにしたい場合に実行してください。

**▼ AIへの指示（以下のテキストをコピーして送信してください）**
```text
以下の作業を行って、アンケート機能を完全に削除してください。

1. src/app/api/chat/route.js から `ENABLE_DECEMBER_SURVEY` フラグの定義を削除してください。
2. src/app/api/chat/route.js から `SURVEY_PROMPT` 定数の定義を削除してください。
3. src/app/api/chat/route.js の `SYSTEM_PROMPT` の定義を、フラグによる分岐がない単純な形（`const SYSTEM_PROMPT = BASE_SYSTEM_PROMPT;` のような形、あるいは直接代入）に戻してください。
4. src/app/page.js のデフォルトメッセージに含まれる「（期間限定でアンケートも実施中！...）」という文言を削除してください。
```
